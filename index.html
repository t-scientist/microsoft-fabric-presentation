Demanda vs. Arquitetura Proposta

Como a solução atende de forma nativa e segura os macro-requisitos solicitados.

Requisitos

Problema Bases Descentralizadas

Múltiplas planilhas Excel com diversas abas distribuídas por gerência, armazenadas de forma dispersa em SharePoint e OneDrive.

Problema Manutenção Estrutural

Alterações estruturais (como inclusão de colunas) exigem ajustes manuais repetitivos em múltiplos arquivos, quebrando relatórios.

Problema Inconsistência de Dados

Falta de validação gera replicação de estruturas com alto risco de divergência entre as bases que alimentam os relatórios.

Problema Dependência Direta

Dashboards conectados diretamente a arquivos Excel como fonte primária, gerando lentidão e falhas frequentes de atualização.

LGPD Controle por Perfil

Restringir a visualização de dados e o controle de acesso conforme perfis (Analista, Supervisor, Coordenador, Gerente e Analytics) e áreas.

LGPD Rastreabilidade

Exigência de log de auditoria completo para identificar quem alterou, quando alterou e qual dado foi modificado.

Performance Concorrência

Garantia de alta performance e disponibilidade ininterrupta para múltiplos usuários consultando e atualizando dados simultaneamente.

Governança Alterações Estruturais

Controle de permissões para modificação da estrutura das bases restrito de forma exclusiva ao time de Analytics.

Integração Integração com Sistemas

Sincronização nativa com Active Directory (AD), SharePoint Lists e integrações potenciais com Totvs, Vulcano e Docusign.

Solução Técnica

Entrada Familiar Foco em Adoção (SharePoint/OneDrive)

As equipes continuam utilizando a interface conhecida do Excel armazenada no SharePoint/OneDrive, garantindo zero fricção e rejeição.

Consolidação Fabric Data Factory Pipelines

Ingestão automatizada de arquivos das áreas para OneLake. O pipeline monitora e valida a estrutura, travando cargas inconsistentes.

Banco Único Azure SQL Serverless / Delta Tables

Consolidação em banco relacional sob demanda. Resolve a duplicidade, estabiliza a modelagem e suporta alta concorrência com performance.

Segurança Azure AD & Row-Level Security (RLS)

Autenticação via AD e filtragem automática de linhas no banco e no Power BI. O usuário vê apenas os dados vinculados à sua área/perfil.

Auditoria Versionamento & Logs de Pipelines

A correlação entre o histórico nativo do SharePoint e logs de processamento no Azure assegura o rastreio completo de edições.

Governança Azure DevOps & Permissões RBAC

Toda a modelagem do banco e relatórios fica no Git do DevOps. A publicação é automatizada, bloqueando edições diretas em produção.

Integrações Azure Functions Serverless

Microsserviços serverless baseados em eventos conectam sistemas como Totvs, Vulcano e Docusign com segurança via chaves no Azure Key Vault.
